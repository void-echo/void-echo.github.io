Hexo blog for beginners

发表于 2021-06-11 Valine：

从零开始搭建一个博客需要多长时间?

大约1整天.

需要做哪些准备?

  • 安装Node.js

  • 安装Hexo.js

  • 安装git (不是Github不是github)

    长这个样子. 我们需要用git bash命令行.

  • 会用git.

  • 熟悉Markdown以及Markdown软件.

    Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

    Markdown 语言在 2004 由约翰·格鲁伯（英语：John Gruber）创建。

    Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。

    Markdown 编写的文档后缀为 .md, .markdown。

    快速链接:

    Markdown语法教程

    Typora下载官网(最流行的Markdown编辑器)

  • 购买一个域名

    在国内, 域名是需要备案的, 平添了不少麻烦.

    所以我推荐使用godaddy 来注册域名(点击可以直达官网).

  • 有一个Github账号

  • 大概 4+ 小时的准备时间. 会遇到各种奇怪的麻烦.

  • 耐心

大概流程

  • 新建一个Github的Page, 一个.io项目

  • 配置hexo

  • 配置主题

  • 发布文章

  • 在合适的时候把网站迁移到自己的域名下面. 需要去网址控制台配置 DNS .

    实际上这一步是非必须的, 但是如果不做, 网站名字就只能是

    xxxxx.github.io

    配置github page 和 你的网站的DNS服务之后, 便可以把你的网址变为 xxx.com

  • 发布文章哈哈哈
